CustomDrive Mission 1 Session

Mission 1 runs the typed route first, then the Pi starts the camera, loads the selected AI model, searches class 1/2 for pickup, grips and lifts the held class, then searches class 3/4 for drop-off based on what it is holding.

Auto sequence: load starting position when Mission 1 starts. When the Pi locates a pickup target (class 1/2) it loads grip ready. When the target is in the forward path and Y is below the configured threshold, it loads grip then grip and lift. At the drop-off target (class 3/4 depending on the held class) it loads release, reverses, then restarts pickup search.
