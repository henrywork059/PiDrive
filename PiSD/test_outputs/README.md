# PiSD Test Outputs

This folder is for small files generated by service test scripts, such as camera smoke-test JPEG captures and motor calibration summaries.

Common generated paths:

```text
test_outputs/camera_service_frame.jpg
test_outputs/camera_color/
test_outputs/camera_settings/
test_outputs/motor_channels/summary.json
```

It is safe to delete generated files here between tests.
