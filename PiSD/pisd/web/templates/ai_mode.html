AI model

Model file

Upload a .keras or .tflite model exported from piTrainer, then load it for preview/drive. PiSD reads piTrainer's named steering and throttle outputs.

For Pi driving, .tflite models need a TFLite runtime installed in the same Python environment that runs PiSD. If Runtime shows TFLite missing, run the shown install command on that Pi and restart PiSD.

Supported: .keras, .tflite, .h5, .onnx, .pt. Runtime support is built in for piTrainer .keras/.tflite models.

Safety

Output limiter

Throttle values use the same full-scale command range as Manual Drive: 0.00 to 1.00. The safety limiter still clamps AI output before it reaches the motors. Reverse steering uses Option A: keep the same steering sign when throttle is negative.
